# Adversarial Generation and Explainable Detection of AI-Enabled Phishing Emails
This repository contains the implementation of my Msc Project investigating AI-generated phishing emails and machine learning detection methods.

## Objectives
- Analyse LLM-generated phishing emails
- Train phishing detection models
- Apply explainable AI techniques
- Evaluate detection performance


## Technologies 
- Python
- Google Colab
- Scikit-learn
- HuggingFace Transformers
- LIME / SHAP

## Project Structure
- datasets/ - Phishing datasets
- notebooks/- experimentation notebooks
- models/ - trained models
- results/- evaluation outputs
